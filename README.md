# simple-pt
Simple "peacetime" script for FiveM RP servers which disables firing and shows status.

INSTALLATION:
- drag simple_pt folder into your resources folder
- add "ensure simple_pt" to your server.cfg

USE: 
- type '/pt' in chat to toggle pt
